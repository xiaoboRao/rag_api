# RAG API 15 节课课程大纲

## 课程定位

这套课程以当前仓库为主教材，目标不是只会把一个 RAG Demo 跑起来，而是循序渐进地理解一个工程化 RAG 检索服务的核心组成：

- 文档如何进入系统
- 文本如何被切块和加元数据
- Embedding 如何生成
- 向量库如何抽象和切换
- 检索 API 如何设计
- 鉴权、回滚、批处理、测试、部署如何落地

这个仓库主要覆盖的是 RAG 的“索引与检索层”，不是完整的“答案生成层”。课程会刻意保留这个边界，让你先把底层打牢，再扩展到完整 RAG。

## 学习路径

这套课程采用“两层结构”：

- 上层是 5 个学习单元，用来建立整体层次感
- 下层是 15 节课，用来保证学习过程足够细

这 15 节课是严格递进的，不建议跳着学。
